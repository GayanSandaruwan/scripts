setup powershell to run scripts
1. press windows + x, open the windows powershel (admin)
2. paste this without quatation "Set-ExecutionPolicy unrestricted"
3. enter y to accept

running the script
1. right click on the script (req1.ps1)
2. choose Run with Powershell
3. Select the input file
4. Output will be generated automatically in the same directory with -out appeneded to the file name
