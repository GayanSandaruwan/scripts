1. usage of wifi1.sh
first open the wifi.sh in a text editor and fill the password field.
bash wifi1.sh "ssid of network" "new password of network"

2. usage of wifi2.sh
first open the wifi.sh in a text editor and fill the "ssid of network", "new password of wifi", "password of admin" fields.
bash wifi2.sh

3. 2. usage of wifi3.sh
first open the wifi.sh in a text editor and fill the "ssid of network", "new password of wifi" fields.
second open a terminal in the directory where the script is located and issue following command
sudo chmod +s wifi3.sh
bash wifi2.sh
